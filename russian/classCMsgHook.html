Detailed Description

Класс для обработки хук-сообщений для окон и их родительско-дочерних связей.

Этот класс предоставляет функционал для присоединения и отсоединения хуков к/от окон. Он управляет взаимодействием между хук-сообщениями и связанными окнами.
