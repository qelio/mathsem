Detailed Description

Диалоговое окно для замены переменных в формуле.

Этот класс предоставляет функционал для управления и замены переменных в формуле. Оно включает списки для отображения переменных и свободных переменных, комбинированные списки для выбора имен и методы для обработки логики замены переменных.
