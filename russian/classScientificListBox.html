Detailed Description

Класс настраиваемого списка, используемый для отображения научных элементов со специальным форматированием для математических символов и индексов.

Этот класс расширяет CListBox и позволяет кастомизировать отрисовку и сортировку элементов списка, в частности для отображения математических символов и выражений.
