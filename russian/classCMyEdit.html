Detailed Description

Пользовательский класс элемента управления редактирования, расширяющий CEdit.

Этот класс предоставляет функционал для обработки пользовательского контекстного меню для элемента управления редактирования. Он включает различные обработчики сообщений для конкретных действий, связанных с контекстным меню.
