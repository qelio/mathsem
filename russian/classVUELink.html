Detailed Description

Класс, представляющий связь между двумя узлами VUE в окружении VUE.

Этот класс используется для моделирования соединений между узлами, включая графические свойства, такие как положение, размер, стиль обводки и цвет. Он предоставляет методы для установки и управления связями между различными типами сущностей, такими как узлы и формулы.
