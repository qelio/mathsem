Detailed Description

Класс, представляющий XML-узел в CXMLArchive.

Этот класс предоставляет методы для манипуляции отдельными XML-узлами, такими как получение дочерних узлов, установка данных и создание новых узлов. Его можно использовать для взаимодействия с XML-узлами в архиве, как с родительскими, так и с дочерними узлами.
