\doxysubsection{Detailed Description}
A class representing an XML node in the \doxylink{classCXMLArchive}{CXMLArchive}. 

This class provides methods for manipulating individual XML nodes, such as retrieving child nodes, setting data, and creating new nodes. It can be used to interact with XML nodes within an archive, either as a parent or child node.
