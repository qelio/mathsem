\doxysubsection{Detailed Description}
A class representing a link between two VUE nodes in the VUE environment. 

This class is used to model connections between nodes, including graphical properties such as position, size, stroke style, and color. It provides methods to set and manage links between different types of entities, such as nodes and formulas.
