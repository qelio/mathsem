\doxysubsection{Detailed Description}
Represents a semantic string for rendering mathematical expressions with styling. 

The \doxylink{classSemanticString}{Semantic\+String} class is responsible for preparing and rendering a string that contains mathematical expressions with various styles such as subscripts, superscripts, and special symbols.
